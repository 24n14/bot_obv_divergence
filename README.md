## 🧠 Описание

**OBV Divergence Trading Bot** — это автоматизированный торговый алгоритм, который:

- 🔍 **Детектирует дивергенции** между ценой и объемом (OBV)
- 📊 **Анализирует тренды** с помощью EMA и ATR индикаторов
- 🎲 **Открывает позиции** при подтверждении дивергенции
- 🛑 **Управляет рисками** через Stop Loss и Take Profit
- 📈 **Ведет логи** всех операций в файл `bot_history.log`

### Принцип работы

1. **Сканирование дивергенций**: Бот ищет расхождения между движением цены и изменением объема
2. **Вычисление уверенности**: Использует 6 факторов для оценки качества сигнала
3. **Открытие позиции**: Входит в рынок, когда уверенность превышает установленный порог
4. **Управление позиции**: Использует трейлинг-стоп для защиты прибыли

## 🔧 Требования

### Системные требования
- Python 3.8+
- pip (менеджер пакетов Python)
- Интернет-соединение (обязательно!)

### Необходимые библиотеки
- `ccxt` — взаимодействие с биржами
- `TA-Lib` — технический анализ
- `python-dotenv` — управление переменными окружения
- `pybit` — API Bybit для трейлинг-стопов
